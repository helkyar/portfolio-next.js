Este proyecto ofrece una herramienta ligera y accesible para la gestión de datos
en línea, permitiendo a los usuarios acceder a sus hojas de cálculo desde
cualquier dispositivo con conexión a internet. Basado en React con Typescript,
Vitest para pruebas y Sonner para mensajes dinámicos, ha mejorado mis
habilidades en desarrollo web y me ha permitido explorar arquitecturas y
patrones de diseño.

## Tecnologías

- React con Typescript
- Vitest para pruebas
- Sonner para mensajes de toast

## Características

- **Gestión de Archivos**: Subir y Descargar hojas
- **Selección con Ratón y Teclado**: Seleccionar columnas, filas y áreas de
  celdas para edición, eliminación, arrastrar y soltar o gestión del
  portapapeles
- **Menú Contextual Personalizado**: Para gestionar el portapapeles y la edición
  de matrices
- **Crear Referencias**: Referenciar celdas y detectar referencias cíclicas
- **Edición Dinámica de Matrices**: Añadir y eliminar columnas y filas
  actualizando cualquier referencia de celda existente
- **Accesibilidad Completa con Teclado**
- **Responsivo**

## Mejoras Futuras

No hay fin cuando se trata de crear una hoja de cálculo completamente
funcional...

- **Más características**: Una de las más interesantes a explorar es referenciar
  celdas de otras hojas así como permitir la personalización de estilos de
  celdas (color, fuente, formato, ...). Redimensionar celdas y combinarlas,
  seleccionar múltiples columnas o filas y permitir arrastrarlas e insertarlas
  con animación de vista previa, deshacer y rehacer...
- **Características más dinámicas**: Permitir insertar imágenes y generar
  gráficos
- **Analíticas Avanzadas**: Integrar analíticas más sofisticadas para entender
  mejor el comportamiento del usuario y la interacción con el contenido.

## Conclusión

Construir este proyecto ha sido una experiencia enriquecedora que me permitió
profundizar en la gestión de datos matriciales. El uso de principios como SOLID
y DRY en conjunto con algunos patrones me ha permitido crear una plataforma que
no solo es funcional, sino también escalable y fácil de mantener. A medida que
continúo creciendo y evolucionando como desarrollador, espero mejorar este
sitio.
