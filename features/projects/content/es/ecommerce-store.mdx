Esta es una tienda ecommerce full-stack construida con NextJs y Stripe. Utiliza
el paquete `@stripe/stripe-js` para manejar pagos y el paquete
`@next/bundle-analyzer` para analizar el tamaño del bundle.

## Características

- **Pagos con Stripe**: Acepta pagos usando la API de Stripe.
- **Catálogo de Productos**: Muestra productos en un diseño de cuadrícula.
- **Carrito de Compras**: Añade productos al carrito de compras y realiza el
  pago.
- **Autenticación de Usuarios**: Permite a los usuarios registrarse e iniciar
  sesión.
- **Historial de Pedidos**: Visualiza el historial de pedidos y rastrea envíos.

## Tecnologías

- **NextJs**: Un framework de React para construir sitios web estáticos y
  dinámicos.
- **Stripe**: Una plataforma de procesamiento de pagos para negocios en línea.
- **Tailwind CSS**: Un framework CSS de utilidad para construir diseños
  personalizados.
- **Vercel**: Una plataforma en la nube para sitios estáticos y funciones
  serverless.

## Comenzando

Para comenzar con este proyecto, puedes clonar el repositorio e instalar las
dependencias:

```bash
git clone
cd ecommerce-store
npm install
```

Una vez instaladas las dependencias, puedes ejecutar el servidor de desarrollo:

```bash
npm run dev
```

El servidor de desarrollo se iniciará en `http://localhost:3000`.

## Despliegue

Para desplegar este proyecto, puedes usar Vercel. Puedes vincular tu repositorio
de GitHub a Vercel y desplegar el proyecto con un solo clic.

## Conclusión

Esta tienda ecommerce es un gran ejemplo de cómo puedes construir una aplicación
full-stack con NextJs y Stripe. Al integrar la API de Stripe con NextJs, puedes
crear una experiencia de compra fluida para tus clientes y gestionar los pagos
de manera segura y eficiente.
