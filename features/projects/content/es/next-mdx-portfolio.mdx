Este es un sitio web de portafolio construido con NextJs y MDX. Utiliza el
paquete `@next/mdx` para renderizar contenido MDX en páginas de NextJs.

## Características

- **Soporte MDX**: Escribe contenido en formato MDX.
- **Resaltado de Sintaxis**: Los bloques de código tienen resaltado de sintaxis.
- **Componentes Personalizados**: Usa componentes personalizados en tus archivos
  MDX.
- **Diseño Responsivo**: El sitio web es responsivo y funciona en todos los
  dispositivos.

## Tecnologías

- **NextJs**: Un framework de React para construir sitios web estáticos y
  dinámicos.
- **MDX**: Un formato de archivo que te permite escribir JSX en tus archivos
  Markdown.
- **Tailwind CSS**: Un framework CSS de utilidad primero para construir diseños
  personalizados.
- **Vercel**: Una plataforma en la nube para sitios estáticos y funciones
  serverless.

## Instalación

Para comenzar con este proyecto, puedes clonar el repositorio e instalar las
dependencias:

```bash
git clone
cd next-mdx-portfolio
npm install
```

Una vez instaladas las dependencias, puedes ejecutar el servidor de desarrollo:

```bash
npm run dev
```

El servidor de desarrollo se iniciará en `http://localhost:3000`.

## Despliegue

Para desplegar este proyecto, puedes usar Vercel. Puedes vincular tu repositorio
de GitHub a Vercel y desplegar el proyecto con un solo clic.

## Conclusión

Este sitio web de portafolio es un gran ejemplo de cómo puedes usar NextJs y MDX
para construir un sitio web moderno con contenido dinámico. Al combinar el poder
de NextJs y MDX, puedes crear sitios web interactivos y atractivos que son
fáciles de mantener y actualizar.
