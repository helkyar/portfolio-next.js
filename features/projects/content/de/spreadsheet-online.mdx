Dieses Projekt bietet ein leichtes und zugängliches Werkzeug zur Verwaltung von
Daten online, das es den Benutzern ermöglicht, von jedem Gerät mit
Internetverbindung auf ihre Tabellen zuzugreifen. Basierend auf React mit
Typescript, Vitest für Tests und Sonner für dynamische Nachrichten, hat es meine
Fähigkeiten in der Webentwicklung verbessert und mir ermöglicht, Architekturen
und Designmuster zu erkunden.

## Technologien

- **React mit TypeScript**: Eine JavaScript-Bibliothek zum Erstellen von Benutzeroberflächen, erweitert mit TypeScript für statische Typüberprüfung.
- **Sonner**: Eine Bibliothek zum Anzeigen von Toast-Nachrichten.
- **Vitest**: Ein blitzschnelles Unit-Test-Framework, das von Vite angetrieben wird.
- **Playwright**: Eine Node.js-Bibliothek zur Automatisierung von Tests in Chromium, Firefox und WebKit mit einer einzigen API.

## Funktionen

- **Dateiverwaltung**: Hochladen und Herunterladen von Tabellen
- **Maus- und Tastaturauswahl**: Auswahl von Spalten, Zeilen und Zellenbereichen
  zur Bearbeitung, Löschung, Drag & Drop oder Zwischenablageverwaltung
- **Benutzerdefiniertes Kontextmenü**: Zur Verwaltung der Zwischenablage und
  Matrixbearbeitung
- **Referenzen erstellen**: Zellen referenzieren und zyklische Referenzen
  erkennen
- **Dynamische Matrixbearbeitung**: Hinzufügen und Löschen von Spalten und
  Zeilen unter Berücksichtigung bestehender Zellreferenzen
- **Volle Tastaturzugänglichkeit**
- **Responsive**

## Installation

Um mit diesem Projekt zu beginnen, kannst du das Repository klonen und die
Abhängigkeiten installieren:

```bash
git clone
cd spreadsheet
npm install
```

Nachdem die Abhängigkeiten installiert sind, kannst du den Entwicklungsserver
starten:

```bash
npm run dev
```

Der Entwicklungsserver wird unter `http://localhost:5173` gestartet.

## Bekannte Fehler

Derzeit wird die Bereichsauswahl mit der Maus in Firefox nicht unterstützt. Die
gleiche Funktionalität kann mit der Tastatur oder durch gleichzeitiges Klicken
beider Maustasten erreicht werden.

## Zukünftige Verbesserungen

Es gibt kein Ende bei der Erstellung einer voll funktionsfähigen
Tabellenkalkulation...

- **Mehr Funktionen**: Eine der interessantesten Funktionen, die es zu erkunden
  gilt, ist das Referenzieren von Zellen aus anderen Tabellen sowie die
  Personalisierung von Zellstilen (Farbe, Schriftart, Format, ...). Zellengröße
  ändern und Zellen kombinieren, mehrere Spalten oder Zeilen auswählen und mit
  Vorschauanimation ziehen und einfügen, Rückgängig und Wiederherstellen...
- **Dynamischere Funktionen**: Ermöglichen das Einfügen von Bildern und das
  Erstellen von Grafiken
- **Erweiterte Analysen**: Integration fortschrittlicherer Analysen, um das
  Benutzerverhalten und die Interaktion mit dem Inhalt besser zu verstehen.
