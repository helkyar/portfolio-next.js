## Technologien

- **Next.js**: Ein React-Framework zum Erstellen von serverseitig gerenderten und statischen Webanwendungen.
- **React mit TypeScript**: Eine JavaScript-Bibliothek zum Erstellen von Benutzeroberflächen, verbessert mit TypeScript für statische Typüberprüfung.
- **Vercel/Postgres**: Vercel für das Deployment und Postgres für die Datenbankkonnektivität.
- **Tremor**: Eine Bibliothek zum Erstellen von Grafiken und Visualisierungen.
- **Sonner**: Eine Bibliothek zum Anzeigen von Popup-Nachrichten.

## Funktionen

- **E-Mail-Verifizierung**: Sicherstellen, dass Benutzer ihre E-Mail-Adressen verifizieren können, um ihre Konten zu schützen.
- **Drittanbieter-Login**: Ermöglichen Sie Benutzern, sich mit Diensten von Drittanbietern wie Google oder Facebook anzumelden.
- **CRUD für Rechnungen und Kunden**: Implementieren Sie Create-, Read-, Update- und Delete-Operationen zur Verwaltung von Rechnungs- und Kundendaten.
- **Abfragen mit mehreren Parametern**: Ermöglichen Sie komplexe Abfragen mit mehreren Parametern, um Daten effizient zu filtern und zu sortieren.
- **Dynamische Grafiken**: Erstellen Sie Grafiken, die sich in Echtzeit basierend auf Benutzerinteraktionen und Datenänderungen aktualisieren.
- **Paginierung**: Implementieren Sie Paginierung, um große Datensätze zu verwalten und die Leistung zu verbessern.
- **Full-Stack-Projekt**: Ein umfassendes Projekt, das sowohl die Front-End- als auch die Back-End-Entwicklung abdeckt.
- **Responsives Design**: Sicherstellen, dass die Anwendung vollständig responsiv ist und auf allen Geräten perfekt funktioniert.

## Installation

Um mit diesem Projekt zu beginnen, können Sie das Repository klonen und die Abhängigkeiten installieren:

```bash
git clone
cd dashboard
npm install
```

Nachdem die Abhängigkeiten installiert sind, können Sie den Entwicklungsserver starten:

```bash
npm run dev
```

Der Entwicklungsserver wird unter `http://localhost:3000` gestartet.
