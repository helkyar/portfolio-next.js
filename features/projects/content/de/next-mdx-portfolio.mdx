Dies ist eine Portfolio-Website, die mit NextJs und MDX erstellt wurde. Sie
verwendet das `@next/mdx`-Paket, um MDX-Inhalte in NextJs-Seiten zu rendern.

## Funktionen

- **MDX-Unterstützung**: Schreiben Sie Inhalte im MDX-Format.
- **Syntax-Hervorhebung**: Codeblöcke werden mit Syntax-Hervorhebung angezeigt.
- **Benutzerdefinierte Komponenten**: Verwenden Sie benutzerdefinierte
  Komponenten in Ihren MDX-Dateien.
- **Responsive Design**: Die Website ist responsiv und funktioniert auf allen
  Geräten.

## Technologien

- **NextJs**: Ein React-Framework zum Erstellen statischer und dynamischer
  Websites.
- **MDX**: Ein Dateiformat, das es Ihnen ermöglicht, JSX in Ihren
  Markdown-Dateien zu schreiben.
- **Tailwind CSS**: Ein Utility-First-CSS-Framework zum Erstellen
  benutzerdefinierter Designs.
- **Vercel**: Eine Cloud-Plattform für statische Websites und serverlose
  Funktionen.

## Erste Schritte

Um mit diesem Projekt zu beginnen, können Sie das Repository klonen und die
Abhängigkeiten installieren:

```bash
git clone
cd next-mdx-portfolio
npm install
```

Sobald die Abhängigkeiten installiert sind, können Sie den Entwicklungsserver
starten:

```bash
npm run dev
```

Der Entwicklungsserver wird unter `http://localhost:3000` gestartet.

## Bereitstellung

Um dieses Projekt bereitzustellen, können Sie Vercel verwenden. Sie können Ihr
GitHub-Repository mit Vercel verknüpfen und das Projekt mit einem einzigen Klick
bereitstellen.

## Fazit

Diese Portfolio-Website ist ein großartiges Beispiel dafür, wie Sie NextJs und
MDX verwenden können, um eine moderne Website mit dynamischen Inhalten zu
erstellen. Durch die Kombination der Leistungsfähigkeit von NextJs und MDX
können Sie interaktive und ansprechende Websites erstellen, die einfach zu
warten und zu aktualisieren sind.
