Ceci est une boutique en ligne full-stack construite avec NextJs et Stripe. Elle
utilise le package `@stripe/stripe-js` pour gérer les paiements et le package
`@next/bundle-analyzer` pour analyser la taille du bundle.

## Fonctionnalités

- **Paiements Stripe**: Accepter les paiements en utilisant l'API Stripe.
- **Catalogue de produits**: Afficher les produits dans une disposition en
  grille.
- **Panier d'achat**: Ajouter des produits au panier et passer à la caisse.
- **Authentification des utilisateurs**: Permettre aux utilisateurs de
  s'inscrire et de se connecter.
- **Historique des commandes**: Voir l'historique des commandes et suivre les
  expéditions.

## Technologies

- **NextJs**: Un framework React pour construire des sites web statiques et
  dynamiques.
- **Stripe**: Une plateforme de traitement des paiements pour les entreprises en
  ligne.
- **Tailwind CSS**: Un framework CSS utilitaire pour créer des designs
  personnalisés.
- **Vercel**: Une plateforme cloud pour les sites statiques et les fonctions
  serverless.

## Pour commencer

Pour commencer avec ce projet, vous pouvez cloner le dépôt et installer les
dépendances :

```bash
git clone
cd ecommerce-store
npm install
```

Une fois les dépendances installées, vous pouvez lancer le serveur de
développement :

```bash
npm run dev
```

Le serveur de développement démarrera à `http://localhost:3000`.

## Déploiement

Pour déployer ce projet, vous pouvez utiliser Vercel. Vous pouvez lier votre
dépôt GitHub à Vercel et déployer le projet en un seul clic.

## Conclusion

Cette boutique en ligne est un excellent exemple de la façon dont vous pouvez
construire une application full-stack avec NextJs et Stripe. En intégrant l'API
Stripe avec NextJs, vous pouvez créer une expérience d'achat fluide pour vos
clients et gérer les paiements de manière sécurisée et efficace.
