Ceci est un site web portfolio construit avec NextJs et MDX. Il utilise le
package `@next/mdx` pour rendre le contenu MDX dans les pages NextJs.

## Fonctionnalités

- **Support MDX**: Écrire du contenu au format MDX.
- **Surlignage de syntaxe**: Les blocs de code sont surlignés.
- **Composants personnalisés**: Utilisez des composants personnalisés dans vos
  fichiers MDX.
- **Design réactif**: Le site web est réactif et fonctionne sur tous les
  appareils.

## Technologies

- **NextJs**: Un framework React pour construire des sites web statiques et dynamiques.
- **MDX**: Un format de fichier qui vous permet d'écrire du JSX dans vos fichiers Markdown.
- **Tailwind CSS**: Un framework CSS utilitaire-first pour créer des designs personnalisés.
- **Vercel**: Une plateforme cloud pour les sites statiques et les fonctions serverless.
- **Playwright**: Une bibliothèque Node.js pour automatiser les tests dans Chromium, Firefox et WebKit avec une seule API.

## Installation

Pour commencer avec ce projet, vous pouvez cloner le dépôt et installer les
dépendances :

```bash
git clone
cd next-mdx-portfolio
npm install
```

Une fois les dépendances installées, vous pouvez lancer le serveur de
développement :

```bash
npm run dev
```

Le serveur de développement démarrera à `http://localhost:3000`.

## Déploiement

Pour déployer ce projet, vous pouvez utiliser Vercel. Vous pouvez lier votre
dépôt GitHub à Vercel et déployer le projet en un seul clic.

## Conclusion

Ce site web portfolio est un excellent exemple de la façon dont vous pouvez
utiliser NextJs et MDX pour construire un site web moderne avec du contenu
dynamique. En combinant la puissance de NextJs et MDX, vous pouvez créer des
sites web interactifs et engageants qui sont faciles à maintenir et à mettre à
jour.
