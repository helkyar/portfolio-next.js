NextJs es un poderoso framework de React que facilita la creación de
aplicaciones React renderizadas en el servidor. Esta publicación te introducirá
a los conceptos básicos de NextJs.

## ¿Qué es NextJs?

NextJs es un framework de React que te permite construir aplicaciones React
renderizadas en el servidor con facilidad. Proporciona una serie de
características de serie, como la renderización del lado del servidor, la
generación de sitios estáticos y las rutas de API, que facilitan la creación de
aplicaciones React rápidas y escalables.

```jsx
export default function Home() {
  return <h1>Hello, NextJs!</h1>
}
```

## ¿Por qué usar NextJs?

NextJs es una excelente opción para construir aplicaciones React renderizadas en
el servidor porque proporciona una serie de características que facilitan la
creación de aplicaciones rápidas y escalables. Algunas de las características
clave de NextJs incluyen:

- Renderización del lado del servidor: NextJs te permite renderizar tus
  componentes de React en el servidor, lo que puede mejorar el rendimiento y el
  SEO.
- Generación de sitios estáticos: NextJs puede generar archivos HTML estáticos
  para tus páginas en el momento de la compilación, lo que puede mejorar el
  rendimiento y reducir los costos de alojamiento.
- Rutas de API: NextJs te permite crear rutas de API que se pueden usar para
  obtener datos de APIs externas o realizar lógica del lado del servidor.
