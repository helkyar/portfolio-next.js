NextJs est un puissant framework React qui facilite la création d'applications
React rendues côté serveur. Cet article vous présentera les bases de NextJs.

## Qu'est-ce que NextJs ?

NextJs est un framework React qui vous permet de créer des applications React
rendues côté serveur avec facilité. Il offre un certain nombre de
fonctionnalités prêtes à l'emploi, telles que le rendu côté serveur, la
génération de sites statiques et les routes API, qui facilitent la création
d'applications React rapides et évolutives.

```jsx
export default function Home() {
  return <h1>Hello, NextJs!</h1>
}
```

## Pourquoi utiliser NextJs ?

NextJs est un excellent choix pour créer des applications React rendues côté
serveur car il offre un certain nombre de fonctionnalités qui facilitent la
création d'applications rapides et évolutives. Certaines des principales
fonctionnalités de NextJs incluent :

- Rendu côté serveur : NextJs vous permet de rendre vos composants React sur le
  serveur, ce qui peut améliorer les performances et le référencement.
- Génération de sites statiques : NextJs peut générer des fichiers HTML
  statiques pour vos pages au moment de la construction, ce qui peut améliorer
  les performances et réduire les coûts d'hébergement.
- Routes API : NextJs vous permet de créer des routes API qui peuvent être
  utilisées pour récupérer des données à partir d'API externes ou effectuer une
  logique côté serveur.
